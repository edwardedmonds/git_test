JPG

JPG images are great for photos and pictures with gradients. They can handle large color palettes without huge file sizes.

GIF

GIF images are great for simple animations but they have a limited color palette.

PNG

PNG images are great for everything that is not a photo or animated. They also can handle opacity and don't have color palette limitations.

SVG

JPG, GIF, and PNG, are all pixel-based formats or raster image formats. SVG is a vector based format which means SVG images can be scaled up and down without loss of quality. They are great for responsive design and are great for all the use cases that PNG images are. Use them whenever you can.

When using SVG images that contain text convert text fields into outlines.
